Toronto Neighbourhood Demographic Analysis Dashboard
Project Overview

This project analyzes demographic patterns across Toronto neighbourhoods using data visualization and interactive dashboards. The goal is to explore how population and age distribution vary across different neighbourhoods in Toronto.

The dashboard provides insights into:

Total population across neighbourhoods

Age distribution (0–14, 15–64, and 60+)

Demographic comparisons between neighbourhoods

The analysis is presented through an interactive Power BI dashboard.

Tools & Technologies

Python – Data cleaning and transformation

Microsoft Excel – Data preparation and pivot analysis

Power BI – Interactive dashboard and visualization

GitHub – Project hosting and version control

Dashboard Features

Total Population KPI – Displays total population based on selected neighbourhood.

Population by Neighbourhood Chart – Compares population across neighbourhoods.

Age Distribution Chart – Shows demographic age groups within neighbourhoods.

Interactive Filter – Allows users to select and explore individual neighbourhoods.

Key Insights

Population distribution varies significantly across Toronto neighbourhoods.

Some neighbourhoods have a higher proportion of working-age residents.

Age demographics differ across communities, which may influence housing demand and urban planning.
